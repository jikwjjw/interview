+ 业务接口
```
package pattern.普通简单工厂模式;

public interface Sender {
    public void send();
}
```
+ 业务1
```
public class EmailSender implements Sender {
    @Override
    public void send() {
        System.out.println("发送邮件");
    }
}
```
